Smart Energy Meter with accurate Monitoring and Billing System

Project information

  • programming skills: Php, Javascript, Html, Css, and Mysqli
  • Final year Project
  • Project date: 01 March, 2020
  • Project Status : Completed
  • Project URL: energymeter88.000webhostapp.com

This project is a smart, low-cost solution to existing non-smart metering systems. This project involves the use of internet of a things (IoT) to provide easy process monitoring of energy consumption and proper billing system. Also with the use of (IoT) the consumer can monitor his consumed reading through webpage. This system enables the electricity department to read the meter readings monthly without a person visiting each house. This system continuously records the reading of the meter and reading can be displayed on webpage to the consumer on request. This system also can be used to disconnect the power supply of the house automatically (ON/OFF) when needed. And also it involved the use of internet and the concept of IoT by which the electricity distribution company, as well as users, remain updated with the current consumed units, changing the present problems faced by the electricity board and the user.
